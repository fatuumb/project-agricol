# 🌱 Smart Agriculture Monitoring avec IA

## 🚀 Description
Projet de monitoring agricole intelligent basé sur l'Intelligence Artificielle.

Objectif : aider les agriculteurs à optimiser la production grâce à la data.

---

## 📊 Fonctionnalités
- 📈 Analyse des données agricoles
- 🤖 Prédiction avec Machine Learning
- 🚨 Détection d’anomalies
- 📊 Dashboard interactif (Streamlit)

---

## 🛠️ Technologies utilisées
- Python
- Streamlit
- Pandas / NumPy
- Scikit-learn
